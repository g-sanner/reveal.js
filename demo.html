Project Overview

What is Open Library?
An open source, editable library catalog.

What does Open Library do?
It's goal is to create a singular web page to host every book ever published.

Who uses Open Library?
Open Library is used by:
1. Readers, to track their reading habits
2. Librarians, to gather data on new books
3. Developers, for access to their API
